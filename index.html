A design kit

AiraKit is a lightweight, static design system built with pure HTML, CSS and JavaScript. Drop it into any project — no build tools, no dependencies, no framework required.

Get Started GitHub

Features

Everything you need,
nothing you don't.

AiraKit gives you a solid foundation of layout utilities, components and design tokens — all in plain CSS classes.

⚡

Zero dependencies

No npm, no build step, no bundler. Just link the CSS and you're good to go.

🎨

Design tokens

Colors, spacing and typography defined as CSS custom properties. Easy to override and theme.

🌙

Dark mode

Automatic dark mode via prefers-color-scheme, plus a manual toggle with localStorage persistence.

📐

Layout utilities

Flexbox-based classes for alignment, spacing, gaps and borders — composable and predictable.

🧩

Ready-made components

Buttons, cards, alerts, forms and more — copy the HTML and it just works.

📱

Responsive

Mobile-first breakpoints built in. Works on any screen size out of the box.

Compatibility

Works with any stack.

Because AiraKit is pure static CSS and JS, it fits into any project without friction.

Flask Django FastAPI Laravel Ruby on Rails Next.js Nuxt Astro Plain HTML WordPress Joomla Any backend →

Get Started

Add it to your project.

Download the kit and link the CSS in your HTML. No installation needed.

base.html · Flask example 
<!-- In your base template -->
<head>
  <link rel="stylesheet" href="{{ url_for('static', filename='css/base.css') }}">
  <link rel="stylesheet" href="{{ url_for('static', filename='css/components.css') }}">
  <link rel="stylesheet" href="{{ url_for('static', filename='css/themes.css') }}">
  <script src="{{ url_for('static', filename='js/main.js') }}" defer></script>
</head>

<!-- Then use classes anywhere -->
<div class="cont txtc p1">
  <h1>Hello from AiraKit</h1>
  <a href="/start" class="btn_act">Get Started</a>
</div>
